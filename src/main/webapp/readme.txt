Setup Instructions
1. Import the 645HW1.war file into Eclipse. by creating a new Dynamic web project. 
2. Then import the war file into the newly created file by right-clicking on the project and selecting import. browser the war file and select it to import. 
3. War file will be imported into the Eclipse 
4. Start the Tomcat server (The WAR file is compatible with Tomcat version 10.1)
5. Right click on the project >> Run As >> Run on server
6. Select a server from the list of servers and click next.
6. In the browser access: http://localhost:<portnumber>/645HW1/

You can use this website's step-by-step instructions for setting up a static website in AWS to build an AWS S3 bucket.
https://docs.aws.amazon.com/AmazonS3/latest/userguide/HostingWebsiteOnS3Setup.html

URL of S3 Bucket/Homepage: http://swe-645-hw1.s3-website-us-east-1.amazonaws.com
The link to the survey form page is at the bottom of the homepage.
